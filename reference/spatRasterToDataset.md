# spatRasterToDataset

Convert a SpatRaster to a SpatRasterDataset.

## Usage

``` r
spatRasterToDataset(x)
```

## Arguments

- x:

  SpatRaster with names of the form "y\[0-9\]+..\[varname\]"

## Value

SpatRasterDataset
