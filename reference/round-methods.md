# Round-method for MAgPIE objects

Round-method for MAgPIE-objects respectively. Works exactly as for
arrays.

## Usage

``` r
# S4 method for class 'magpie'
round(x, digits = 0)
```

## Arguments

- x:

  a magpie object

- digits:

  integer indicating the number of decimal places (round) or significant
  digits (signif) to be used. Negative values are allowed.

## Methods

- x = "magpie":

  works as round(x) for arrays.
